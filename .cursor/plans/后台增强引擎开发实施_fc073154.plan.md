## 约束与规范

- 后端 Case 三件套：实体/接口/实现 + contracts.md 同步 + `.http` 测试文件
- 前端 Case：组件 + i18n + `npm run build` 通过
- 每个 Case 完成后 `dotnet build` 0 错误 0 警告
- 写操作 API 要求 `Idempotency-Key` + `X-CSRF-TOKEN`
- 数据库操作禁止循环内逐条执行
- 新路由使用 `api/v1` 版本前缀
