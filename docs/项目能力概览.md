# 项目能力概览

## 平台定位

- 面向等保2.0合规建设的安全支撑平台与基础设施支撑平台。
- 多租户架构，按租户隔离数据与权限。

## 关键能力

### 身份与权限

- 账号登录与令牌签发（JWT）。
- 用户、角色、权限、菜单与部门管理。
- 角色授权与菜单授权。

### 资产与告警

- 资产台账查询与新增。
- 告警查询与分页检索。

### 审计

- 行为审计记录查询。
- 审批关键动作写入审计日志。

### 审批流

- 流程定义管理与发布。
- 发起流程、查询实例、取消实例。
- 待办任务审批与驳回。
- 抄送记录与已读标记。
- 审批运行时操作（撤回、转办、加签、打回修改、退回任意节点、撤销同意等）。
- 部门负责人配置，用于审批人计算。

### 工作流

- 工作流定义与实例查询。
- 启动、挂起、恢复、终止工作流实例。
- 外部事件发布。

### 安全与合规

- JWT + 证书双认证体系。
- CORS 白名单与 HTTP 日志记录。
- 租户上下文中间件与安全策略配置。
- 密码策略与锁定策略的集中配置。

## 架构与技术

### 架构分层

- Domain / Application / Infrastructure / WebApi 四层清晰分离。
- 以领域为核心的模块化边界：Assets、Audit、Alert、Approval、Workflow。

### 技术栈

- 后端：.NET 10、SqlSugar + SQLite、AutoMapper、FluentValidation、NLog、IdGen。
- 前端：Vue 3、Vite、Ant Design Vue、Router。
