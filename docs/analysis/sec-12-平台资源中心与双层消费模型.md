# SEC-12 平台资源中心与双层消费模型（含 SEC-51/52）

## 1. 任务信息

- Linear：`SEC-12`（`[P1] 建立平台资源中心承接治理类能力`）
- 覆盖子任务：`SEC-51`、`SEC-52`
- 所属里程碑：`P1 模型与边界收口`

## 2. 平台治理资源清单（SEC-51）

| 资源类型 | 必须平台治理原因 | App 侧可见能力 |
|---|---|---|
| 模型（LLM/Embedding） | 统一密钥、额度、合规策略 | 选择与绑定 |
| 连接器/OAuth | 统一安全审核与凭据托管 | 安装与配置实例 |
| 插件定义 | 统一审核、版本治理 | 安装、启用、参数配置 |
| 模板与市场资源 | 统一上架审核、分发策略 | 复制到应用工作台 |
| 发布策略 | 统一审批、回滚与审计 | 发起发布请求 |

## 3. 双层消费模型

```mermaid
flowchart LR
  P[平台注册/审核] --> C[可消费目录]
  C --> A[App 安装/绑定/配置]
  A --> R[Runtime 使用]
```

## 4. SEC-52：资源中心 IA 分组

| 分组 | 主入口 | 二级入口 | 不应承载 |
|---|---|---|---|
| 模型中心 | `/console/resources/models` | 供应商、模型版本、配额 | 应用 Prompt 编辑 |
| 连接器中心 | `/console/resources/connectors` | 连接器类型、OAuth | 应用内调试页 |
| 插件市场 | `/console/resources/plugins` | 插件定义、上架审核 | 应用实例参数页 |
| 模板中心 | `/console/resources/templates` | 行业模板、流程模板 | 租户运行数据 |
| 发布中心 | `/console/resources/releases` | 审核策略、发布规则 | 运行态任务列表 |
| 开放平台 | `/console/resources/open-platform` | PAT、Webhook、回调策略 | 应用页面设计器 |

## 5. 平台与 App 边界挂载表

| 能力 | 平台侧 | App Workspace |
|---|---|---|
| 资源注册 | 是 | 否 |
| 资源审核 | 是 | 否 |
| 资源安装 | 否 | 是 |
| 资源绑定配置 | 否 | 是 |
| 运行使用 | 否 | 通过 Runtime |

## 6. 高风险混层区域

| 区域 | 风险 | 治理策略 |
|---|---|---|
| AI 平台菜单直接放编辑器 | 平台与应用边界模糊 | 编辑器下沉到 App Workspace |
| 低代码中心含运行监控 | 构建态/运行态混层 | 监控回迁治理中心 |
| 开放平台入口分散 | 安全策略无法统一 | 聚合到资源中心开放平台分组 |
