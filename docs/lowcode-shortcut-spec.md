# 低代码快捷键清单（lowcode-shortcut-spec）

> 状态：M00 预创建 stub。
> 范围：M04 画布 + M07 Studio 壳的完整快捷键清单（≥ 40 项），冲突检测，用户自定义键位策略。

## 章节占位（≥ 40 项快捷键，由 M04 / M07 完整列出）

- §1 编辑类：撤销（Ctrl+Z）/ 重做（Ctrl+Shift+Z 或 Ctrl+Y）/ 复制（Ctrl+C）/ 剪切（Ctrl+X）/ 粘贴（Ctrl+V）/ 同位粘贴（Ctrl+Shift+V）/ 删除（Delete / Backspace）/ 全选（Ctrl+A）/ 取消选择（Esc）
- §2 对齐类：左对齐 / 居中对齐 / 右对齐 / 顶端对齐 / 居中（垂直）/ 底端对齐
- §3 分组类：分组（Ctrl+G）/ 解组（Ctrl+Shift+G）/ 锁定 / 显隐 / 居中
- §4 移动类：键盘移动 1px（方向键）/ Shift+10px / 包裹容器 / 解包 / 前置 / 后置 / 上移一层 / 下移一层
- §5 视图类：缩放（Ctrl+= / Ctrl+- / Ctrl+0）/ 适应屏幕 / 实际大小 / 网格 / 参考线
- §6 全局类：Tab 切换 Tab / Ctrl+S 保存 / Ctrl+Shift+P 命令面板 / Ctrl+/ 帮助
- §7 冲突检测：与浏览器内置快捷键冲突的处理策略
- §8 用户自定义键位：覆盖与重置策略

> 完整清单由 M04 / M07 落地。
